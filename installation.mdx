## Installation
To use TaillyUI in your project, run one of the following commands in your terminal:
<CodeGroup>
```bash npm 
npm i tailly
```
```bash yarn
yarn add tailly
```
```bash pnpm
pnpm add tailify
```
</CodeGroup>
